## Linear Equation Solver (C++)

### Description

This program reads a system of linear equations and generates the augmented matrix representation.

Features :

- Parses equations dynamically

- Supports implicit coefficients

- Command interface

- Prints formatted matrix

### Requirements

- C++ Compiler (g++)

- Standard C++11 or higher

- Compile
    g++ LinearEquationCalc.cpp -o solver
    Run
./solver

### Commands

- num_vars

- equation i

- quit
